# SME Market Analysis Project
Bu projede, 14 yıllık bankacılık tecrübemi istatistiksel yöntemlerle birleştirdim.

### Neler Yaptım?
* **EDA (Keşfedici Analiz):** Verideki eksik değerleri temizledim ve aykırı değerleri ayıkladım.
* **Regresyon Analizi:** X değişkeni ile Y değişkeni arasındaki ilişkiyi ölçtüm.
* **Sonuç:** Veriler, KOBİ'lerin [X] döneminde [Y] eğilimi gösterdiğini kanıtlıyor.

### Kullanılan Araçlar
* MS Excel (Pivot Tables, Data Analysis Toolpak)
